AgroSIG CADET - Versión final para entrega

Incluye:
- Visor web con Leaflet.
- Capas GeoJSON: cultivos, bosque, pastos, invernaderos, paso de agua y borde CADET.
- Panel para activar/desactivar capas.
- Ficha de información al hacer clic.
- Registro local de monitoreos.

Cómo ejecutar:
1. Abrir esta carpeta en VS Code.
2. Abrir index.html.
3. Presionar Go Live.
